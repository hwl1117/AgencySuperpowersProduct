# VideoBrain - 短视频智能知识库产品架构

## 产品概述
VideoBrain 是一款结合 Agency-Agents 和 Superpowers 的智能产品，能够通过短视频平台链接一键识别视频内容并概括优化成知识库。

## 核心功能流程
```
短视频链接 → 视频下载 → 音频提取 → 语音转文字 → 视觉关键帧分析 → AI概括优化 → 知识库存储 → 搜索检索
```

## 技术架构

### 1. 后端服务 (Python FastAPI)
- **视频下载服务**: 支持抖音、快手、B站、YouTube Shorts等平台
- **音频提取服务**: 使用FFmpeg提取音频
- **语音转文字服务**: 集成OpenAI Whisper API
- **视觉分析服务**: 提取关键帧并进行图像识别
- **AI概括服务**: 使用GPT-4进行内容概括和优化
- **知识库管理服务**: 向量数据库存储和检索

### 2. 前端界面 (React + TypeScript)
- **URL输入界面**: 支持批量输入视频链接
- **处理状态显示**: 实时显示处理进度
- **知识库浏览**: 分类浏览和搜索
- **导出功能**: 支持多种格式导出

### 3. 数据库设计
- **SQLite**: 存储元数据、用户信息
- **ChromaDB**: 向量数据库，存储语义嵌入
- **Redis**: 缓存和任务队列

### 4. AI能力 (Superpowers)
- **内容理解**: 视频内容深度分析
- **智能概括**: 自动生成结构化摘要
- **知识提取**: 关键信息提取和分类
- **语义搜索**: 基于向量的智能搜索

## 产品特性
1. **一键操作**: 粘贴链接即可处理
2. **多平台支持**: 覆盖主流短视频平台
3. **智能分析**: AI驱动的内容理解
4. **知识管理**: 结构化知识库
5. **持续学习**: 系统不断优化处理能力

## 部署架构
- Docker容器化部署
- 支持本地和云部署
- 水平扩展能力
